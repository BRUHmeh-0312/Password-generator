# Password generator 💻

## About 🚀
A simple python project that can generate a fully random password that can be used by you freely. 🔥

This project might not be the best one ever, but I have been working on it for some time now, and considering that I'm still a beginner,
so I think it was a good project to enchant my programming skills.

It's also very useful to generate strong passwords!

Enter the number of digits you want your password to have.
Then enter the number of passwords to generate.

This will print the number of password you enetred with the amount of digits you wrote.
